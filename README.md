# 🔮 AI 面相大师

基于人工智能的面相分析应用，融合中国传统面相学与现代AI技术。

## ✨ 功能特点

- 📸 **简单上传**：支持拖拽或点击上传面部照片
- 🤖 **AI智能分析**：基于智谱AI视觉模型进行面相解读
- 📊 **详细报告**：包含五官、三停、十二宫位等完整分析
- 🎯 **运势预测**：事业、财运、感情、健康四大运势
- 🍀 **开运指南**：幸运颜色、数字、方位建议

## 🛠️ 技术栈

- **框架**: Next.js 15 + React 19
- **样式**: Tailwind CSS
- **AI**: 智谱 GLM-4V-Plus 视觉模型
- **部署**: Vercel

## 📚 面相学知识

### 五官
- 耳（采听官）：关系长寿、智慧
- 眉（保寿官）：关系健康、地位
- 眼（监察官）：关系意志力、心性
- 鼻（审辨官）：关系财富、中年运
- 口（出纳官）：关系幸福、晚年运

### 三停
- 上停：15-30岁运势
- 中停：31-50岁运势
- 下停：51岁后运势

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 添加智谱API密钥

# 启动开发服务器
npm run dev
```

## 📝 环境变量

| 变量名 | 说明 |
|--------|------|
| ZHIPU_API_KEY | 智谱AI API密钥 |
